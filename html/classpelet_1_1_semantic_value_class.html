Detailed Description

This is the low-level building block that the parser creates as it is traversing the source code. The parser's job will be to convert these low-level building blocks into higher-level class, method, function, and variable declarations.

using pointers here because Bison will create a union with this type as one of the values; and unions values have to be Plain Old Data types (which UnicodeString is not). Also, this cannot have a non-trival constructor; Init() should be called right after object construction. Be sure to init and free with the ParserObserverClas; that way memory can be cleaned up correctly.
