Detailed Description

This class will represent one PHP Expression. An expression is either

  • a variable ($abc)
  • a scalar (constant)
  • an operation between 2 variables (arithmetic, boolean, bitwise)
  • an operation between 1 variables and another expression
  • an operation of 1 variable (increment, decrement, cast, error suppression, clone)
  • a "new" object construction
  • a function call
  • a variable assignment of one variable to an expression

Member Data Documentation

std::vector<ExpressionClass> pelet::ExpressionClass::CallArguments

In the case of function calls; this is the arguments of the function call (each of which could be the results of other function calls).

std::vector<UnicodeString> pelet::ExpressionClass::ChainList

In the case of a variable; this is the list of properties / methods that were successively invoked. For example; the expression "$this->name->toString()" will have 3 items in the chain list "$this" "->name" and "->toString()".

UnicodeString pelet::ExpressionClass::Comment

The comment that is attached to the function call / variable.

UnicodeString pelet::ExpressionClass::Lexeme

In the case of a scalar; this is the token text (actual string or int value). In the case of a variable; this is the variable name. In the case of a variable that chains object operations ($this->name->toString) then the lexeme is the value of the first variable in the chain ("$this" in the example above).

QualifiedNameClass pelet::ExpressionClass::Name

In the case of a function call; this is the name of the function that was called. In the case of a NEW_CALL, then name is the name of the class that was instantiated. For all other types; this is empty.
