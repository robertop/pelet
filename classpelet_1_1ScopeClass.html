Detailed Description

This class represents a scope of a variable: ie the containing namespace, class, and function name.

UnicodeString pelet::ScopeClass::FullyQualify ( const pelet::QualifiedNameClass &  name,
const pelet::QualifiedNameClass &  declaredNamespace 
) const

Calculate the fully qualified name from a namespace name, taking aliases into account.

example code:

namespace MyProject {

use My as Another;

class MyClass implements \Framework\Base, Another { ... }

}

then names can be: "MyClass" or "\\Framework\Base", "Another" declaredNamespace is always "MyProject"

Parameters:
namethe name of the class / function / constant. this name may contain an alias, or may contain sub-namespaces
declaredNamespacethe namespace in which the class class / function / constant is defined in. this is always fully qualified.
