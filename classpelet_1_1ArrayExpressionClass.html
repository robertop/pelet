Detailed Description

This class represents an array expression. The ArrayPairs contains the list of key-value pairs of an array. For example, for the expression: array('one' => 1, 'two' => 2) then ArrayPairs will contain 2 items: the pair "one", 1 (a scalar expression as the key and a scalar expression as the value) and "two" (a scalar expression as the key and a scalar expression as the value) Note that if the source code does not define keys then this key will be NULL as well; for example array(1, 2) will not produce any array keys (ArrayPairs will still have 2 items, but the keys will be empty).
