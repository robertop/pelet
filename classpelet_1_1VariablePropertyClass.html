Detailed Description

The VariablePropertyClass represents a single object property of a variable. For example, the variable $this->myFunct()->name->first will have 4 VariableProperty objects: $this myFunct() name first

Member Data Documentation

pelet::ExpressionClass* pelet::VariablePropertyClass::ArrayAccess

When this property is an array access, then this will be the expression inside the array access operator for example: the property $users[$name] will have ArrayAccess = VariableClass instance. This class will NOT own this pointer. note that his may be NULL for empty array access ie. $arr[] = 1;

std::vector<pelet::ExpressionClass*> pelet::VariablePropertyClass::CallArguments

If this property is a method, then this vector will contain the function call arguments. This class will NOT own any of the pointers

bool pelet::VariablePropertyClass::IsArrayAccess

bool TRUE if this property or method contains a array access for example for $users['name'] this will be 2 VariableProperty instances, the first with Name = "users", IsArrayAccess = false, and the second with Name = "", IsArrayAccess = true and ArrayAccess = a ScalarExpressionClass instance
